# Usedoze

Site institucional da Usedoze, criado com HTML, CSS e JavaScript.

## Objetivo

Apresentar serviços de criação de sites, landing pages e soluções web simples, com visual moderno e responsivo.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Como rodar localmente

Abra o arquivo `index.html` no navegador ou use uma extensão como Live Server no VS Code.

## Deploy no GitHub Pages

Este repositório já inclui um workflow em `.github/workflows/pages.yml`.

Depois de enviar para o GitHub:

1. Vá em **Settings**
2. Clique em **Pages**
3. Em **Source**, escolha **GitHub Actions**
4. Aguarde o deploy finalizar

O site ficará disponível em um endereço parecido com:

```txt
https://angelosantos824.github.io/usedoze/
```

## Observação

O formulário abre uma mensagem pronta no WhatsApp. Para usar seu número real, altere o número em `script.js` e os links `wa.me` nas páginas HTML.
